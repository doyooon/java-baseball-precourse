# java-baseball-precourse

### 기능 요구사항 목록
- [ ] 컴퓨터가 랜덤 3자리수 생성  
- [ ] 사용자에게 수 입력받기  
- [ ] 입력받은 수와 비교하여 힌트 생성  
  - [ ] 스트라이크 : 같은 수가 같은 자리에 있을 때  
  - [ ] 볼 : 같은 수가 다른 자리에 있을 때  
  - [ ] 낫싱 : 같은 수가 전혀 없을 때  
- [ ] 힌트 출력하기  
- [ ] 사용자가 잘못된 값을 입력한 경우 `IllegalArgumentException` 발생시키기  
- [ ] 사용자가 수를 맞췄을 때, 게임을 다시 할 건지 물어보기
